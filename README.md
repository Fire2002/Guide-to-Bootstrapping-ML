# Guide-to-Bootstrapping-ML
A practical guide and curated resources for individuals and small teams looking to bootstrap ML projects from idea to deployment with limited resources.

Bootstrap topics covered:
- Cross-Validation Bootstrapping
- Basic Bootstrapping
- Bayesian Bootstrapping
- Bias-Corrected Bootstrapping
- Fast-Bootstrapping
- Jackknife-Bootstrapping
- Leave-One-Out (LOO) Bootstrapping
- Moving-Block Bootstrapping
- Non-Parametric Bootstrapping
- Parametric Bootstrapping
- Standard Bootstrapping
- Stratified Bootstrapping
- Studentized Bootstrapping
- Wild Bootstrapping
